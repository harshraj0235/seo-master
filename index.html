How to Install & Use

Get started in less than 60 seconds.

1

Download the Extension

Download the latest .zip file from the GitHub Releases page and extract the folder to a safe location on your computer.

2

Enable Developer Mode

Open Chrome and navigate to chrome://extensions/. In the top right corner, toggle Developer mode to ON.

3

Load Unpacked

Click the Load unpacked button and select the folder you extracted in Step 1. The SEO Master icon will appear in your toolbar!

4

Open the Side Panel

Navigate to any webpage you want to audit. Click the SEO Master extension icon in your toolbar to slide open the beautifully integrated Side Panel.
